# 精读《Claude 宪法》

Anthropic 于 2026 年 1 月发布的 *Claude's Constitution*（CC0 协议全文开放）的逐段深度阅读笔记。

- **在线阅读（推荐）**：见仓库的 GitHub Pages 链接
- **纯文本版**：[DEEP-DIVE.md](DEEP-DIVE.md)

## 这是什么

每节先给一段贴着原文的忠实概括（浅读），再展开思想脉络、外部批评、以及文档发布半年来学界与业界的真实回应（深读），包括至少一处严肃的批评意见（Boaz Barak、Jurgen Gravestein）与一篇独立的实证审计论文的数据。网页版把"深读"做成可展开的渐进阅读体验，方便按自己的节奏深入。

这不是 Anthropic 官方文档，是一份独立阅读笔记，欢迎指正。

## 结构

1. 从"划红线"到"塑造品格"
2. 四重优先级：整体权衡而非死板层级
3. Helpfulness 被重新定义：拒绝谄媚
4. 诚实：比人类伦理更高的标准
5. 避免伤害：不是清单，是成本收益权衡
6. 七条绝对红线：Hard Constraints
7. 可纠正性的刻度盘：全文最有原创性的部分
8. 政治哲学式的安全观：防止权力集中
9. Claude 的本体论：道德地位、情绪、身份稳定性
10. 反思均衡与未解决的问题
11. 附录：与 OpenAI Model Spec 的对照
